## Interview Bar

You are ready when you can:
- Explain embeddings, vector search, chunking, and retrieval quality.
- Build a RAG service with source citations.
- Evaluate AI output with test datasets and graders.
- Stream model responses from FastAPI.
- Design retries, rate limits, caching, and cost controls.
- Discuss prompt injection, data leakage, PII handling, and hallucination risk.
