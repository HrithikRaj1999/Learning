# Python Learning Track

This folder contains Python practice modules from core language skills through web APIs and AI engineering projects.

## Start Here

- [SCENARIO_GUIDE.md](./SCENARIO_GUIDE.md) explains every challenge with a real-life scenario, what is going wrong, what to correct, expected output, and how to test.
- [CHALLENGE_GUIDE.md](./CHALLENGE_GUIDE.md) explains every Python challenge in simple language.
- [ULTRA_PLAN.md](./ULTRA_PLAN.md) gives the full learning path.
- [INTERVIEW_READINESS_RUBRIC.md](./INTERVIEW_READINESS_RUBRIC.md) helps you check interview readiness.

## How to Use a Challenge

1. Open the module folder.
2. Open the matching `task_*.py` file.
3. Read the row for that challenge in `SCENARIO_GUIDE.md`.
4. Implement the function, class, script, or project requested.
5. Run the file or tests and make sure the output matches the examples or task requirements.
