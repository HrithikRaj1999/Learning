# DSA Practice Track

This folder contains the curated DSA problem list by pattern and priority.

## Start Here

- [Final_DSA_20_Questions_Per_Section.md](./Final_DSA_20_Questions_Per_Section.md) is the main problem list.
- [SCENARIO_GUIDE.md](./SCENARIO_GUIDE.md) explains every problem with the interview scenario, what goes wrong with naive code, expected output, and how to test.
- [CHALLENGE_GUIDE.md](./CHALLENGE_GUIDE.md) explains what is expected for every problem and what your output should be.

## How to Use a Challenge

For every problem, write the brute force idea, optimized idea, data structure, complexity, edge cases, and dry run before coding. Your final output should be an accepted solution for the problem's required return type plus a short explanation.
